## **量化规则设计**

### **模型 1 (S\_)**

总分 0-100

|     |     |     |
| --- | --- | --- |
| 项目  | 权重  | 量化规则 |
| 1\. 均线收缩/收敛 | 0-20 | 取 MA10、MA20、MA50，计算过去 20 日均线间距的标准差，越小得分越高（最小值满分）。 |
| 2\. 枢轴邻近/突破 | 0-15 | 过去 60 日最高收盘价作为枢轴，现价 ≥ 枢轴的 97% 且 ≤ 103% 得满分，偏离度越大扣分。 |
| 3\. 成交量(20/60日&斜率) | 0-20 | 近 20 日均量 / 近 60 日均量 越大越高（放量得分高）；20 日量能斜率向上加分。 |
| 4\. 口袋妖怪 | 0-15 | 过去 50 日内，至少 3 次出现大阳线（收盘>开盘2%以上且量>20日均量1.5倍）则满分，按比例得分。 |
| 5\. 均线结构 | 0-10 | MA10 > MA20 > MA50 且低点逐级抬高（最近3个波谷），满足全部满分。 |
| 6\. 上方套牢供给 | 0-10 | 现价距离过去 250 日最大成交量区间越近，得分越低，反之越高。 |
| 7\. 收盘强度 | 0-5 | 当日收盘位置在当日最高-最低区间的百分比（80%以上满分）。 |
| 8\. 低位反包信号 | 0-5 | 当日为低位反包K线（长下影+收阳且量放大）则满分。 |

### **模型 2 (B\_shape)**

总分 0-100

|     |     |     |
| --- | --- | --- |
| 项目  | 权重  | 量化规则 |
| 1\. VCP 压缩 | 0-30 | 波动幅度连续 3 次缩小（高低点振幅依次下降），且量能递减则满分。 |
| 2\. RSI 动量 | 0-30 | RSI14 在 50-70 区间且向上斜率 >0.5 则高分，RSI>70 但未背离满分。 |
| 3\. 动量 | 0-20 | 20 日收益率 / 波动率，值越大得分越高。 |
| 4\. 均线收敛 | 0-10 | 同模型 1 的均线收缩，但权重较轻。 |
| 5\. 口袋妖怪 | 0-10 | 同模型 1 的定义。 |

* * *

### **计算方式**

	周期：最近 60 个交易日（大约 2 个月）
	
	评分：每个因子按比例换算到权重范围，求和得总分。
	
	输出：股票代码 + 股票名称 + 模型1分数 + 模型2分数
