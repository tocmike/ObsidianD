# 【田子方】第143集-真实的人是什么样子

喜马拉雅的朋友，你好！我是傅佩荣。

这一集的主题是：真实的人是什么样子。

我们进展到《庄子‧田子方》篇第21，这一篇也是很多寓言跟重言，其中还出现不少高明的人物。像东郭顺子、温伯雪子、臧丈人、伯昏无人、孙叔敖、真儒士、真画师等等。里面也提到颜渊跟孔子学习，后来发现“孔子奔逸绝尘”；然后孔子往见老子，才知道“天地之大全”；这真是人外有人，天外有天。这几篇文章的用意，都是鼓励我们领悟大道，以成就《老子‧道德经25》所说的“道大，天大，地大，人亦大”。这样的理想。

这一集所谈的是〈田子方〉篇第一章跟第二章，田子方名字叫做田无择，据说他跟魏文侯都是孔子的学生子夏的学生。所以在这里就出现了田子方跟魏文侯的对话；第二章是温伯雪子上场，他也是一个高手。

我们先看第一章，庄子怎么写的。

“田子方陪坐在魏文侯旁边，谈话时多次称赞谿工。文侯说：“谿工是你的老师吗？”子方说：“不是的，是我的同乡。他的言谈议论都很有道理，所以我称赞他。”文侯说：“那么你没有老师吗？”子方说：“有。”文侯说：“你的老师是谁呢？”子方说：“是东郭顺子。”

文侯说：“那么你为什么没有称赞过他？”子方说：“我的老师为人真诚，外貌一如常人而内心与自然相合，顺应世俗而能保持真实，洁身自处而能包容外物。对于无道的人，就用庄重的态度来开导，使他打消邪念。我哪里有资格称赞他呢！”

子方离去之后，文侯若有所失，整天不说话，然后召来站在身边的臣子，对他们说：“遥不可及啊，禀赋完全的君子！起初我以为圣智的言论、仁义的行为是最高明的，现在我听到还有子方的老师这种人，我的身体好像散开了不想动，嘴巴好像封住了不想说。我过去所学的，简直是土制的假人！魏国真是我的拖累啊！”

从后面魏文侯的说法，可以知道他有可能真的跟儒家学过。==学了之后才发现什么圣智的言论、仁义的行为都不够看==。现在听到田子方所描述的东郭顺子的表现，才受到震撼：身体不想动，嘴巴不想说，过去所学的是土制的假人！结论是什么？魏国变成我的拖累啊！

还记得前面(〈山木20.2-20.3〉)楚国的贤人市南宜僚去见鲁国的国君，说鲁国是您的拖累。现在，魏文侯自己觉悟了：魏国是我的拖累！在这里，所谓的东郭顺子的表现是什么？就是我们一直强调的“==外化而内不化==”。把这句话掌握住就没有问题了。==你看，他为人真诚，外貌一如常人，而内心与自然相合。这不是====“外化而内不化”==吗？“顺应世俗而能保持真实，洁身自处而能包容外物”。非常标准的描述。

田子方跟了东郭顺子这样的老师，所以境界当然是比较高超了。至于他是不是跟魏文侯一样也跟子夏学过，那是一个说法。就算是学过的话，庄子也可以借重他的名字来表达自己的思想。这一集说的真实的人，所指的就是东郭顺子做到“外化而内不化”，是就这一点来说的。

接着，看到〈田子方〉篇的第二章，温伯雪子上场。温伯雪子出现不只一次，将来还会看到列御寇也跟温伯雪子请教过。

在这个地方谈什么？谈温伯雪子经过鲁国的时候，鲁国总有人想向他请教。我们知道鲁国是周公之后，是非常重要的文化古国。那么，温伯雪子对这个就有一点批评的思考。
我们先看看庄子怎么写的。

“温伯雪子前往齐国，中途投宿于鲁国。鲁国有人请求见他，温伯雪子说：“不行，我听说鲁国的君子，明白礼义的形式，却拙于了解人心，我不想见他。”他到了齐国之后，回程也投宿于鲁国，这个人又来请求见他。温伯雪子说：“上次要求见我，现在又要求见我，想必有什么可以启发我的事。”于是出去见了客人，回到房间就叹气。第二天再出去见了客人，回到房间又叹气。

他的仆人说：“每次去见这位客人，回到房间一定叹气，为什么呢？”温伯雪子说：“我早就告诉你了：『鲁国的人明白礼义的形式，却拙于了解人心。』我所见过的这个人，进退有规有矩，举止从容像龙与虎一般；劝告我的态度像儿子对待父亲，开导我的态度像父亲对待儿子，所以我要叹气啊。”

孔子见到温伯雪子时，没说任何话。子路说：“老师早就想见温伯雪子了。现在见了面却不说话，为什么呢？”孔子说：“像他这样的人，视线相接就可以展现大道，也不容许多说什么了。””

在这里我们看到什么？第一个，鲁国是一个有传统的，重视礼乐教化的国家。但是，==他了解的礼跟义的形式，却没有了解人心==。等于是==只注重外在的形式，而忽略了真诚的内心的状况==。所以温伯雪子经过鲁国的时候，鲁国一直有人慕名而来，要跟他见面谈谈话。

他第一次拒绝了，后来想：别人一定要见我，一定有什么要开导我吧！结果见了客人回来就叹气。第二天这个人又来了，温伯雪子回房间又叹气。仆人就忍不住问他：“这怎么回事呢？见了这个人回来就叹气。”

温伯雪子说：“我早就说过了：『鲁国的人了解了外在的形式，却不了解人心内在的状况。』这个人，进退有规有矩，举止从容像龙跟虎一样；他劝告我的时候像儿子对待父亲，开导我的时候像父亲对待儿子，所以我要叹气啊。”

看到没有！很有规矩，讲究礼乐教化，各种人间的规范通通遵守。但是，==我跟你有什么样的关系？你为什么老想去扮演老师的角色，来开导劝告别人==？这是完全浪费力气的事，所以这就是温伯雪子的故事。

最后，孔子上场了，孔子不一样，他显然不是前面那位鲁国人，孔子见了温伯雪子根本不讲话。子路跟在旁边就很好奇，说：“老师，您早就想见这个人了。现在见了面却不说话，怎么回事呢？”孔子说：“像这样的人，视线相接就可以展现大道。”原文说得真是精采“==目击而道存矣==”。眼睛互相看一眼，大道就展现出来。

就好像真正的武侠高手，不用大战三百回合。不必了！见了面，姿势一摆，就知道谁赢谁输了；要不然就知道是不是同一个门派出来的，不必多说话！

这在这里是另外一种境界，因为真正好朋友悟道之后，才能够“相视而笑，莫逆于心”。就是不用说话，说话是多余的。一说话就容易提出各种观点，想出什么理论，然后大家来互相参考辩论一下。这个，庄子早就超越了。

人活在世界上只有一件事：就是要“悟道”。所悟的“道”是万物的来源跟归宿，形成一个整体。大家都在这个整体里面，各人修练各人的，能够悟道的话，就可以很开心地眼睛视线交接，就可以觉得没有问题了。我们都对于“道”有所领悟，知道这是最重要的。

在这一集里面，我们学到什么？真实的人是什么样子。前面所说的那位是东郭顺子；第二章所说的是温伯雪子，都是庄子虚拟的非常具有代表性的人物，都达到了真人、神人这样的境界！

我们从这里面也知道，庄子真的很喜欢拿儒家做为一个跳板，让你知道儒家全部投入世间固然是很辛苦，也非常好心。但是呢，你还是要往上提升！否则，终究离“道”隔了几层，这是非常可惜的事情。
